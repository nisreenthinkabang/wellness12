# แผนของเรา — wellness12

เว็บแอพ (PWA) สำหรับโปรแกรมสุขภาพ 12 สัปดาห์ ใช้ร่วมกันสองคน

- เนื้อหาทั้งหมดถูก **เข้ารหัส AES-GCM** เก็บใน `content.enc.json` — ต้องมีรหัสผ่านจึงจะเปิดอ่านได้
- เพิ่มลงหน้าโฮมโทรศัพท์ได้ (ทำงานแบบแอพ + ใช้แบบออฟไลน์ได้หลังเปิดครั้งแรก)
- ความคืบหน้า (เช็กลิสต์เวท, สติกเกอร์, โทเคน, บันทึกผล) เก็บไว้ในเครื่องเท่านั้น

## เปลี่ยนรหัสผ่าน

```bash
node tools/reencrypt.mjs "รหัสเดิม" "รหัสใหม่"
git add content.enc.json && git commit -m "change passcode" && git push
```

> โน้ต: การเข้ารหัสฝั่งเว็บช่วยกันคนที่ไม่มีรหัสอ่านเนื้อหาได้จริง แต่ความแข็งแรงขึ้นกับรหัสผ่าน — ตั้งรหัสที่เดายาก
